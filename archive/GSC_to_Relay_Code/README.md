# MCU Code

This folder contains the microcontroller application code for the LoRa relay control system.

## Files

- **SenderMCU_to_LoRa1.cpp** - Sender MCU code that sends relay commands via LoRa
- **LoRa2_RecieverMCU_to_Relay.cpp** - Receiver MCU code that receives LoRa commands and controls relays

## Hardware
- XIAO ESP32S3
- E22 LoRa modules
- 6-channel relay board

## Usage
Each file is a complete PlatformIO project source file. Use with the LoRa_Library for full functionality.
